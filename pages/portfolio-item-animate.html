Text, Logo & BannerAnimation

with Adobe Animate

Logo animation

In this case, I designed a website logo during one of my early assignments. It created it with the pen tool on the basis of an jpg image. Although image trace can be faster and produce much realistic image, there is one drawback, namely its relatively big size. Therefore creating a vector image with the pen tool has got a considerable advantage regarding file size, but it can also be manipulated more easily, for instance change colors. As the s logo can be broken up into parts and animated separately, which gives and extra features to the design.

The Zwolsche Vogels text animation

The purpose of this animation was, in the first place, to create a logo animation, but it became et the end a banner animation. Banner animation can be a port of a logo animation; however in this case it is almost purely but not entirely a text animation. The animation, here, can be divided into two parts: the animation of the letters e.g. text animation, and the animation the graphic elements that make up the background. Here mainly masking and scaling are used with classic tween in animate.
