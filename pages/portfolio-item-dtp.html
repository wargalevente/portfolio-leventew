Magazine and booklet design

For this portfolio item, I chose to remake a nature walk brochure. However I cannot reproduce all the images so I placed my own nature images in the brochure. Remaking this brochure involves numerous features of InDesign from Parent Pages, Character, Paragraph en Object Styles to Text Frames and Text Wrap. How to use Parent Pages based on another Parent Page. For instance, the best way to keep the footer and header consistent throughout an InDesign document is to create a separate Parent Page for header and footer and use it as the basis Parent Page for other Parent Pages.
